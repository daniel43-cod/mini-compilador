10. Diagramas de Conway

Los diagramas de Conway representan gráficamente las estructuras sintácticas reconocidas por el analizador sintáctico implementado.

Declaración de variable

flowchart LR
    A([Inicio]) --> B["let | var | const"]
    B --> C["IDENTIFICADOR"]
    C --> D["="]
    D --> E["VALOR"]
    E --> F[";"]
    F --> G([Fin])

Asignación

flowchart LR
    A([Inicio]) --> B["IDENTIFICADOR"]
    B --> C["= | += | -= | *= | /="]
    C --> D["VALOR"]
    D --> E[";"]
    E --> F([Fin])

Impresión con console.log

flowchart LR
    A([Inicio]) --> B["console"]
    B --> C["."]
    C --> D["log"]
    D --> E["("]
    E --> F["VALOR"]
    F --> G[")"]
    G --> H[";"]
    H --> I([Fin])

Condicional IF

flowchart LR
    A([Inicio]) --> B["if"]
    B --> C["("]
    C --> D["CONDICIÓN"]
    D --> E[")"]
    E --> F["{"]
    F --> G["SENTENCIAS"]
    G --> H["}"]
    H --> I([Fin])

Condicional ELSE

flowchart LR
    A([Inicio]) --> B["else"]
    B --> C["{"]
    C --> D["SENTENCIAS"]
    D --> E["}"]
    E --> F([Fin])

Bucle FOR

flowchart LR
    A([Inicio]) --> B["for"]
    B --> C["("]
    C --> D["Inicialización"]
    D --> E[";"]
    E --> F["Condición"]
    F --> G[";"]
    G --> H["Incremento"]
    H --> I[")"]
    I --> J["{"]
    J --> K["SENTENCIAS"]
    K --> L["}"]
    L --> M([Fin])

Bucle WHILE

flowchart LR
    A([Inicio]) --> B["while"]
    B --> C["("]
    C --> D["CONDICIÓN"]
    D --> E[")"]
    E --> F["{"]
    F --> G["SENTENCIAS"]
    G --> H["}"]
    H --> I([Fin])

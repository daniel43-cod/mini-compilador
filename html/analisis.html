6. Análisis Léxico con Código de Ejemplo

Para comprobar el funcionamiento del analizador léxico, se utiliza el siguiente bloque de código escrito en JavaScript.

Código de ejemplo

const edadPersona = 20;

if (edadPersona >= 18) {
    console.log("Acceso concedido. Puede ingresar.");
} else {
    console.log("Acceso denegado. Es menor de edad.");
}

Resultado del análisis

El analizador léxico identifica correctamente palabras reservadas, identificadores, operadores, literales numéricos, literales de cadena y símbolos especiales. Además, cada token generado incluye el lexema encontrado, su categoría léxica y el número de línea donde aparece.

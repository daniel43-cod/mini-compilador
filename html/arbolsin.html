8. Árboles de Análisis Sintáctico

El árbol de análisis sintáctico representa la estructura jerárquica de una sentencia del lenguaje. Cada nodo muestra una parte de la instrucción y sus componentes.

Árbol de Declaración de Variable

Ejemplo:

let edad = 20;
flowchart TD
    A[Declaración]
    A --> B[let]
    A --> C[IDENTIFICADOR: edad]
    A --> D[=]
    A --> E[LITERAL_NUMERICO: 20]
    A --> F[;]

Árbol de Asignación

Ejemplo:

edad = 25;
flowchart TD
    A[Asignación]
    A --> B[IDENTIFICADOR: edad]
    A --> C[=]
    A --> D[LITERAL_NUMERICO: 25]
    A --> E[;]

Árbol de Impresión

Ejemplo:

console.log("Hola mundo");
flowchart TD
    A[Impresión]
    A --> B[console]
    A --> C[.]
    A --> D[log]
    A --> E[(]
    A --> F[LITERAL_CADENA: Hola mundo]
    A --> G[)]
    A --> H[;]

Árbol de Condicional IF

Ejemplo:

if (edad >= 18) {
    console.log("Mayor de edad");
}
flowchart TD
    A[Condicional IF]
    A --> B[if]
    A --> C[(]
    A --> D[Condición]
    D --> E[IDENTIFICADOR: edad]
    D --> F[OPERADOR_RELACIONAL: >=]
    D --> G[LITERAL_NUMERICO: 18]
    A --> H[)]
    A --> I[{]
    A --> J[Impresión]
    J --> K[console]
    J --> L[.]
    J --> M[log]
    J --> N[(]
    J --> O[LITERAL_CADENA: Mayor de edad]
    J --> P[)]
    J --> Q[;]
    A --> R[}]

Árbol de Condicional IF - ELSE

Ejemplo:

if (edad >= 18) {
    console.log("Acceso concedido");
} else {
    console.log("Acceso denegado");
}
flowchart TD
    A[Condicional IF ELSE]

    A --> B[if]
    A --> C[(]
    A --> D[Condición]
    D --> E[edad]
    D --> F[>=]
    D --> G[18]
    A --> H[)]
    A --> I[Bloque IF]

    I --> J[Impresión]
    J --> K[console.log]
    J --> L[LITERAL_CADENA: Acceso concedido]

    A --> M[else]
    A --> N[Bloque ELSE]

    N --> O[Impresión]
    O --> P[console.log]
    O --> Q[LITERAL_CADENA: Acceso denegado]

Árbol de Bucle FOR

Ejemplo:

for(i = 0; i < 5; i++) {
    console.log(i);
}
flowchart TD
    A[Bucle FOR]

    A --> B[for]
    A --> C[(]

    A --> D[Inicialización]
    D --> E[i]
    D --> F[=]
    D --> G[0]

    A --> H[Condición]
    H --> I[i]
    H --> J[<]
    H --> K[5]

    A --> L[Incremento]
    L --> M[i]
    L --> N[++]

    A --> O[)]
    A --> P[Bloque]

    P --> Q[Impresión]
    Q --> R[console.log]
    Q --> S[IDENTIFICADOR: i]

Árbol de Bucle WHILE

Ejemplo:

while(i < 10) {
    console.log(i);
}
flowchart TD
    A[Bucle WHILE]

    A --> B[while]
    A --> C[(]

    A --> D[Condición]
    D --> E[i]
    D --> F[<]
    D --> G[10]

    A --> H[)]
    A --> I[Bloque]

    I --> J[Impresión]
    J --> K[console.log]
    J --> L[IDENTIFICADOR: i]

12. Análisis Sintáctico LL(1)

Se realiza el análisis sintáctico LL(1) para la siguiente sentencia: 


let edad = 20;

Tokens de Entrada

Token
let
IDENTIFICADOR
=
LITERAL_NUMERICO
;
$

Pila Inicial

PROGRAMA $

Proceso de Análisis LL(1)

Paso Pila Entrada Acción
1 <programa> $ let IDENTIFICADOR = LITERAL_NUMERICO ; $ Aplicar <programa> → <sentencia>
2 <sentencia> $ let IDENTIFICADOR = LITERAL_NUMERICO ; $ Aplicar <sentencia> → <declaracion>
3 <declaracion> $ let IDENTIFICADOR = LITERAL_NUMERICO ; $ Aplicar producción de declaración
4 let IDENTIFICADOR = VALOR ; $ let IDENTIFICADOR = LITERAL_NUMERICO ; $ Consumir token "let"
5 IDENTIFICADOR = VALOR ; $ IDENTIFICADOR = LITERAL_NUMERICO ; $ Consumir IDENTIFICADOR
6 = VALOR ; $ = LITERAL_NUMERICO ; $ Consumir "="
7 VALOR ; $ LITERAL_NUMERICO ; $ Aplicar VALOR → LITERAL_NUMERICO
8 LITERAL_NUMERICO ; $ LITERAL_NUMERICO ; $ Consumir número
9 ; $ ; $ Consumir ";"
10 $ $ Aceptar cadena

Resultado

La cadena es aceptada por el analizador sintáctico LL(1), ya que todos los símbolos de entrada coinciden correctamente con las producciones de la gramática y la pila finaliza vacía junto con el símbolo de fin de cadena ($).

4. Diseño de AFD para cada Expresión Regular

Los Autómatas Finitos Deterministas permiten representar gráficamente cómo el analizador léxico reconoce cada categoría léxica del lenguaje JavaScript.

AFD para Identificadores

Expresión regular: [a-zA-Z_][a-zA-Z0-9_]*

stateDiagram-v2
    [*] --> q0
    q0 --> q1: letra o _
    q1 --> q1: letra, dígito o _
    q1 --> [*]: aceptar

AFD para Números Enteros

Expresión regular: \d+

stateDiagram-v2
    [*] --> q0
    q0 --> q1: dígito
    q1 --> q1: dígito
    q1 --> [*]: aceptar

AFD para Números Decimales

Expresión regular: \d+\.\d+

stateDiagram-v2
    [*] --> q0
    q0 --> q1: dígito
    q1 --> q1: dígito
    q1 --> q2: punto .
    q2 --> q3: dígito
    q3 --> q3: dígito
    q3 --> [*]: aceptar

AFD para Literal de Cadena

Expresión regular: ".*?" | '.*?'

stateDiagram-v2

    q2 --> [*]: aceptar

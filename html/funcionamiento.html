Funcionamiento del Analizador Léxico y Sintáctico

Descripción General

El sistema desarrollado permite realizar análisis léxico y sintáctico sobre fragmentos de código escritos en JavaScript. El usuario puede ingresar código fuente y obtener información sobre los tokens generados, tabla de símbolos, errores encontrados y árbol sintáctico.

Pasos para utilizar el sistema

  1. Ingresar un bloque de código JavaScript en el área de texto.
  2. Presionar el botón Analizar Código.
  3. El sistema ejecutará el análisis léxico.
  4. Posteriormente se realizará el análisis sintáctico.
  5. Los resultados serán mostrados en las diferentes secciones de la página.

Resultados Generados

Información General

Muestra la cantidad de líneas procesadas y estadísticas generales del análisis.

Tabla de Tokens

Presenta cada lexema identificado junto con su categoría léxica y número de línea.

Tabla de Símbolos

Muestra los identificadores encontrados durante el análisis.

Control de Errores

Informa errores léxicos y sintácticos indicando el tipo de error y la línea donde ocurrió.

Árbol Sintáctico

Representa gráficamente la estructura sintáctica del código analizado.

Ejemplo de Uso


const edadPersona = 20;

if (edadPersona >= 18) {
    console.log("Acceso concedido");
}

Al analizar este código, el sistema identificará palabras reservadas, identificadores, operadores, literales y construirá el árbol sintáctico correspondiente a la estructura condicional.

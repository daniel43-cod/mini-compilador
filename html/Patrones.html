Lenguaje seleccionado

El lenguaje seleccionado para el análisis es JavaScript, un lenguaje de alto nivel utilizado principalmente para el desarrollo web.

Categorías Léxicas

Categoría Descripción Ejemplo
Palabra reservada Palabras propias del lenguaje que tienen una función definida. let, if, else, for, while
Identificador Nombre asignado a variables o funciones. edad, contador, temperatura
Literal numérico Valores numéricos enteros o decimales. 10, 20, 36.5
Literal cadena Texto encerrado entre comillas simples o dobles. "Hola", 'JavaScript'
Literal booleano Valores lógicos verdadero o falso. true, false

Expresiones Regulares

Categoría Expresión Regular
Identificador [a-zA-Z_][a-zA-Z0-9_]*
Literal numérico \d+(\.\d+)?
Literal cadena ".*?" | '.*?'
Booleano true | false

Estructuras Sintácticas Aceptadas

  • Declaración de variables: let edad = 20;
  • Asignación: edad = 25;
  • Impresión: console.log(edad);
  • Condicional if: if (edad >= 18) { ... }
  • Condicional else: else { ... }
  • Bucle for: for(i = 0; i < 5; i++) { ... }
  • Bucle while: while(i < 10) { ... }
